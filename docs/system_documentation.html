පරිශීලක සහය (User Documentation)

පද්ධති සහය (System Documentation)

 

ඇක්සස් දත්ත ගබඩාවේ වගු (tables), පෝරම (forms) ඇතුලු අංග වෙත පිවිසීමට

යතුරු පුවරුවේ F11 යතුර ඔබන්න එවිට පහත පරිදි වම් පසින් අදාල අංග දිස්වේ.

 

 

මෙහි වගුවල ක්ෂේත්‍ර නම් වෙනස් කළ හැක. අලුත් ක්ෂේත්‍ර ඇතුලත් කළ හැක.

නමුත් වගු අතර සම්බන්ධතා (relationships) බිඳ නොවැටෙන පරිදි මේවා සංස්කරණය කළ යුතුය.

වගු අතර සම්බන්ධතා දැනගැනීමට පහත දැක්වෙන පරිදි Ribbon එකේ Database Tools à Relationships වෙත පිවිසෙන්න.

 

 

එවිට පහත රූපයේ පරිදි ඔබට වගු අතර සම්බන්ධතා දැකගත හැක.

 

මෙම සබඳතා ඔබගේ ඇක්සස් දැනුම අනුව අවශ්‍ය නම් සංස්කරණය කළ හැක. සබඳතා බිඳනොවැටෙන ලෙස වගු යාවත්කාලීන කළ හැක.

 

වම් පස ඇති පදිංචිකරුවන්ගේ දත්ත රැස්කරන Residents වගුව පහත දැක්වේ.

 

Details Form යනු දත්ත ගබඩාව විවෘත කරන අවස්ථාවේදී පෙන්වන ප්‍රධාන පෝරමයයි.

මෙම පෝරමය හෝ අනෙකුත් පෝරම ඔබට අවශ්‍ය පරිදි වෙනස් කරගැනීමට පහත පරිදි Layout View හෝ Design View වෙත පිවිසෙන්න.

 

 

පහත පරිදි පෝරම (forms) වර්ග කර දැක්විය හැක.

1.      Residents Form  පදිංචිකරුවන්ගේ තොරතුරු Residents table ඇසුරෙන් පෙන්වයි.

 

2.      FamilyUnitsForm  පවුල් ඒකකවල තොරතුරු FamilyUnits table සහ Residents table ඇසුරෙන් පෙන්වයි.

 

a.       මෙය තුළ පවුලේ සාමාජිකයන් පෙන්වීමට ResidentsSubForm භාවිත වේ.

 

3.      Campaigns Form  වැඩසටහන්වල තොරතුරු Campaigns table, Campaigns Map table සහ Residents table ඇසුරෙන් පෙන්වයි.

a.       මෙය තුළ අදාල වැඩසටහනට ඇතුලත් කරන පදිංචිකරුවන් පෙන්වීමට CampaignsMapSubForm භාවිත වේ.

 

Audit යන වචනයෙන් ආරම්භ වන වගු සංස්කරණ ඉතිහාසයට අදාල තොරතුරු ස්වයංක්‍රීයව රැස් කරයි.

AuditForm ලෙස අවසන් වෙන පෝරම මෙම තොරතුරු පෙන්වයි.
